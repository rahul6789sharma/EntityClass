JAX-RS

JAX-RS: Java API for RESTful Web Services (JAX-RS) is a Java programming language API that provides support in creating web services according to the Representational State Transfer (REST) architectural pattern. JAX-RS uses annotations, introduced inJava SE 5, to simplify the development and deployment of web service clients and endpoints.

Java API for RESTful Web Services (JAX-RS), is a set of APIs to developer REST service. JAX-RS is part of the Java EE6, and make developers to develop REST web application easily.
