REST Example - Hello Web Service

Jersey is the implementation to develop RESTful web service based on JAX-RS (JSR 311) specification.

We will develop a simple hello world REST web application with Jersey and deploy on tomcat server

Technologies and Tools used to develop restful web service

  1. JDK 1.7
  2. Jersey 1.8
  3. Tomcat 7
  4. Maven 3.1.1
  5. Eclipse Luna


1. Generate Maven Web Project Using Command Line

mvn archetype:generate -DgroupId=com.techtutorial.rest -DartifactId=RestHelloDemo -DarchetypeArtifactId=maven-archetype-webapp -DinteractiveMode=false

OR : Create Maven Project Using Eclipse

Open Eclipse Go to File->New->Maven Project

Click next and provide the details of you project

Select the webApp type Maven project and click next

Provide the Group Id and Artifact Id and click on finish . This will create Maven project directly in Eclipse

5. Create REST Service Class

Inside JAVA directory Create Rest Service HelloRestClassService 

package com.techtutorial.rest;

import javax.ws.rs.GET;
import javax.ws.rs.Path;
import javax.ws.rs.PathParam;
import javax.ws.rs.core.Response;


@Path("/hello")
public class HelloRestService {
	
	@GET
	@Path("/{param}")
	public Response getMsg(@PathParam("param") String msg) {
 
		String output = "Welcome by Jersey : " + msg;
 
		return Response.status(200).entity(output).build();
 
	}

}

6. Edit web.xml

You need to create a Web xml Configuration file which is an XML file and is used to specify Jersey framework servlet for our application.

In web.xml, register “com.sun.jersey.spi.container.servlet.ServletContainer“, and puts your Jersey service folder under “init-param“, “com.sun.jersey.config.property.packages“.

"com.sun.jersey.config.property.packages" Tells Jersey what package to scan for your @Path classes and register those classes.

Note: As you can see our servlet is mapped to /rest/ URI pattern. So the basic structure of the URIs to reach the REST Services used in this example will have the form

http://localhost:8080/RestHelloDemo/rest/..


7. Demo

URL: http://localhost:8080/RestHelloDemo/rest/hello/Hello

In this example, web request from “projectURL/rest/hello/” will match to "HelloRestService", via @Path("/hello").

And the “{any values}” from “projectURL/rest/hello/{any values}” will match to parameter annotated with @PathParam
