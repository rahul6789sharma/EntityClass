Eclipse Kepler SwitchYard Plugin

SwitchYard Tooling is compatible with Eclipse JEE Kepler release. Follow the below steps to install the plug-in for switchYard in Eclipse.

  1. Select Help-->Install New Software... from the main menu bar
  2. Press the Add... button and Add... the following sites
    Name: SwitchYard
    Location: http://download.jboss.org/jbosstools/updates/stable/kepler/integration-stack/aggregate/4.1.4.Final/
  3. Once the software site has been loads, SwitchYard Tools should be displayed in the list of available software. (select the features (SwitchYard) to install, or click the Select All button.) Check the box, Click Next, agree to the license terms, and install.

Verify SwitchYard plugin Installation

Start the Eclipse now you can see that we are having option to create new SwitchYard Project.

Open Eclipse Go to File->New->Project and Search For SwitchYard
